# Quizz JS

````
L'objectif est de créer un quizz en JavaScript.

1. créer l'interface, les blocs, les boutons radio.
2. Styliser le tout.
3. Récupérer les données au clic.
4. Comparer les résultats.
5. Afficher des phrases en fonction des résultats.
6. Changer la couleur des blocs + animation en fonction des résultats.
````

## Langage
* HTML
* SCSS
* JS
